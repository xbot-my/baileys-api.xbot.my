# Baileys API

A REST API service for integrating WhatsApp functionality into applications. Built with [NestJS](https://github.com/nestjs/nest) and [Baileys](https://github.com/WhiskeySockets/Baileys).

## Features

- **HTTP API** - Send messages, manage groups, handle contacts
- **WebSocket Events** - Real-time notifications for messages and updates
- **Direct Protocol** - No browser automation required
- **Database Persistence** - Automatic sync of messages, chats, and contacts
- **Docker Ready** - One-command deployment
